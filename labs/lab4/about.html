About Twicker Mobile:

Twicker Mobile

Twicker Mobile is a simple example in uses of the Twitter API to retrieve a set amount of tweets. In it's current form Twicker Mobile grabs 100 tweets posted within 10 miles radius of Troy, NY and displays them in a cycling loop. The call itself is only made once, on page load, and therefore requires the user to reload the page if I post a tweet after their page loads.

A list of the technolgoies I used:

  • PHP Twitter API class structure developed and released by j7mbo [here]
  • A Google Now card style structure developed by Bennett Feely available [here]
  • Foundation 5 and jQuery (mobile)
